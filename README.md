# Julian Dower Portfolio

A React portfolio rebuilt as a long-form bento feed with soft gradients, editorial cards, and one featured project link for [`wissenwert`](https://github.com/juliandower/wissenwert).

## Scripts

### `npm start`

Runs the portfolio locally at `http://localhost:3000`.

### `npm test -- --watchAll=false`

Runs the route-level test suite once.

### `npm run build`

Builds the production bundle into `build/`.

## Notes

- `/` is the main portfolio experience.
- `/portfolio` redirects to `/` for compatibility.
- `/contact` contains the direct contact links used across the site.
